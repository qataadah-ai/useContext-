React Login Form with Context API
Overview

This is a simple React login form project that demonstrates how to:

Use React Context API to manage global state

Create controlled components for form inputs

Display a welcome message after login

When the user logs in, the username is stored in context, and the app can show "Welcome, [username]" anywhere.

Features

Login form with username and password

State management using React Context

Controlled form inputs with proper reset after submit

Reusable UserContext for global user state
